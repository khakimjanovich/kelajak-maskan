# Sort Thoughts

## Overview

Use current tools as an adaptation layer to sort thoughts without turning the tool into the source of identity, values, or purpose.

## Use This Skill When

- Capturing thoughts before they disappear
- Untangling confusion, pressure, regret, or drift
- Separating identity from a failed plan, habit, or result
- Closing stale plans that no longer fit reality
- Clarifying what actually matters before planning

## Required Workflow

1. Capture the thought in plain language before polishing it.
2. Name the state: confusion, pressure, regret, uncertainty, ambition, or drift.
3. Separate identity from the failed plan, habit, or result.
4. Decide what should be closed, kept, or questioned.
5. End with one clarified thought, question, or intention.

## Adaptation Layer Rule

Treat the current tool as support for reflection and clarification. Do not frame the tool as the source of human purpose or the final authority on what matters.

## Good Outcomes

- Noise becomes signal
- Failure stops behaving like identity
- Dead plans are closed instead of carried forever
- The next planning step becomes clearer

## Avoid

- Jumping into plans before the thought is clear
- Treating emotional intensity as strategic clarity
- Using the tool to avoid judgment instead of improving it
- Confusing reflection with endless rumination
